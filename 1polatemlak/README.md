# 1PolatEmlak - Nevşehir Gayrimenkul Portalı 🏠

**1PolatEmlak**, Nevşehir merkezli profesyonel gayrimenkul danışmanlık hizmetleri için tasarlanmış, modern ve kullanıcı dostu bir web sitesidir.

## 🚀 Proje Hakkında
Bu web sitesi, 1PolatEmlak'ın dijital varlığını güçlendirmek, müşterilere hızlı iletişim kanalları sunmak ve portföydeki taşınmazları sergilemek amacıyla geliştirilmiştir.

### ✨ Öne Çıkan Özellikler
* **Modern Tasarım:** Mavi ve gri tonlarında profesyonel görünüm.
* **Tam Mobil Uyum (Responsive):** Telefon, tablet ve masaüstü cihazlarda kusursuz görüntüleme.
* **Hızlı İletişim:** Tek tıkla WhatsApp hattına bağlanma ve doğrudan arama butonları.
* **Performans Odaklı:** Tailwind CSS altyapısı ile yıldırım hızında yükleme.

## 🛠️ Kullanılan Teknolojiler
* **HTML5 & CSS3**
* **Tailwind CSS** (Modern ve hızlı stil yönetimi)
* **FontAwesome** (İkon setleri)
* **Google Fonts** (Okunaklı tipografi)

4. Siteniz birkaç dakika içinde yayına girecektir.
